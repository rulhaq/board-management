## Getting Started

### First Login

1. Navigate to the application URL
2. Click "Login" or "Sign In"
3. Enter your email and password
4. Click "Login"

If you don't have an account, contact your administrator to create one.

### Password Reset

1. Click "Forgot Password?" on the login page
2. Enter your email address
3. Check your email for the reset link
4. Follow the instructions to set a new password

## Authentication

### Login

- Enter your registered email address
- Enter your password
- Click "Login"

### Two-Factor Authentication (2FA)

If 2FA is enabled:

1. After entering your password, you'll be prompted for a 2FA code
2. Open your authenticator app (Google Authenticator, Authy, etc.)
3. Enter the 6-digit code
4. Click "Verify"

### Logout

1. Click on your profile picture/name in the top right
2. Click "Logout"

## Documents

### Viewing Documents

1. Navigate to "Documents" from the main menu
2. Browse all documents or use the search/filter options
3. Click on a document to view details

### Uploading Documents

**All users** can upload documents:

1. Click "Upload Document" or the "+" button
2. Select file(s) from your computer
3. Enter document details:
   - Title
   - Category
   - Description
   - Tags (optional)
4. Click "Upload"

Documents are automatically processed and stored securely.

### Document Categories

- Policies
- Reports
- Minutes
- Financial
- Legal
- Other

### Downloading Documents

1. Find the document in the list
2. Click the download icon
3. The document will download to your computer

### Deleting Documents

**Admin and Secretary** can delete documents:

1. Find the document
2. Click the delete icon
3. Confirm deletion

## Meetings

### Creating a Meeting

**All users** can create meetings:

1. Navigate to "Meetings"
2. Click "Create Meeting" or "+"
3. Fill in meeting details:
   - Title
   - Date and Time
   - Duration
   - Location/Platform
   - Agenda
   - Attendees
4. Click "Save"

### Viewing Meetings

- **Upcoming**: Future meetings
- **Past**: Completed meetings
- **All**: All meetings

### Calendar Export

1. Find the meeting
2. Click "Download Calendar"
3. Choose format:
   - Outlook (.ics)
   - Google Calendar (.ics)
4. Import into your calendar application

### Editing Meetings

**Admin, Secretary, or Meeting Creator** can edit:

1. Find the meeting
2. Click the edit icon
3. Update details
4. Click "Save"

### Meeting Minutes

1. Open a past meeting
2. Click "Add Minutes"
3. Enter meeting notes
4. Save minutes

## Voting

### Creating a Ballot

**All board members** can create ballots:

1. Navigate to "Voting"
2. Click "Create Ballot"
3. Enter ballot details:
   - Title
   - Description
   - Voting options (minimum 2)
   - Start date
   - End date
   - Anonymous voting (optional)
4. Click "Create"

### Voting on a Ballot

1. Find an active ballot
2. Click "Vote"
3. Select your choice(s)
4. Add comments (optional)
5. Click "Submit Vote"

### Viewing Results

1. Find the ballot
2. Click "View Results"
3. See:
   - Total votes
   - Votes per option
   - Percentage breakdown
   - Individual votes (if not anonymous)

### Ballot Status

- **Draft**: Not yet active
- **Active**: Currently accepting votes
- **Closed**: Voting period ended
- **Finalized**: Results are final

## Members

### Viewing Members

1. Navigate to "Members"
2. See all board members with:
   - Name and photo
   - Position
   - Department
   - Contact information
   - Bio

### Member Profiles

Click on a member to view:
- Full profile
- Contact details
- Specialties
- Activity history

### Adding Members

**Admin only**:

1. Navigate to "Members"
2. Click "Add Member"
3. Fill in member details:
   - Name
   - Email
   - Role (Board Member, Secretary, Admin)
   - Position
   - Department
   - Phone
   - Bio
4. Click "Save"

The member will receive an email invitation.

## Reports

### Generating Reports

1. Navigate to "Reports"
2. Select report type:
   - Activity Report
   - Meeting Summary
   - Voting Results
   - Document Inventory
3. Select date range
4. Click "Generate Report"
5. Download as PDF or CSV

### Report Types

- **Activity Report**: All user activities
- **Meeting Summary**: Meeting attendance and minutes
- **Voting Results**: Ballot results and participation
- **Document Inventory**: All documents with metadata

## Tips & Best Practices

### Security

- Use strong passwords
- Enable 2FA if available
- Don't share your login credentials
- Log out when finished

### Document Management

- Use descriptive titles
- Add appropriate categories
- Include tags for easy searching
- Keep documents organized

### Meetings

- Create meetings well in advance
- Include detailed agendas
- Export to calendar
- Add minutes promptly after meetings

### Voting

- Create clear ballot questions
- Set appropriate voting periods
- Review results before finalizing
- Document decisions

## Troubleshooting

### Can't Login

- Check email and password
- Try password reset
- Contact administrator

### Documents Not Uploading

- Check file size limits
- Verify file format is supported
- Check internet connection
- Contact support if issue persists

### Notifications Not Appearing

- Check notification settings
- Verify email is correct
- Check spam folder
- Refresh the page

### AI Assistant Not Responding

- Check AI provider configuration (Admin)
- Verify API keys are set (Admin)
- Check internet connection
- Try again later
